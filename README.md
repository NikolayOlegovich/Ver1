
# Mobile App Interface Design

This is a React-based mobile contact management application with a modern interface design. The original project is available at https://www.figma.com/design/TMAu3XfQxZnyZDypiTRilc/Mobile-App-Interface-Design.

## Features

- **Statistics Screen** - Contact data quality metrics and analytics
- **Contact Selection** - Search and select contacts from address book
- **Contact Dossier** - Detailed contact information with social media integration
- **Categories Management** - Organize contacts into categories and subcategories

## Quick Start

### Automated Setup (Recommended)
```bash
./deploy.sh
```

### Manual Setup
```bash
npm install
npm run dev
```

### Development Mode
```bash
./start-dev.sh
```

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS
- Radix UI components
- Lucide React icons
