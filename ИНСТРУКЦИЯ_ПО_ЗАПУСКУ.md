# Инструкция по запуску мобильного приложения интерфейса

## Описание проекта

Это React-приложение для управления контактами с мобильным интерфейсом. Приложение включает в себя:

- **Экран статистики** - отображение качества контактных данных и общей статистики
- **Экран выбора контактов** - поиск и выбор контактов из адресной книги
- **Экран досье** - подробная информация о контакте с интеграцией социальных профилей
- **Экран категорий** - организация контактов по категориям и подкатегориям

## Системные требования

### Обязательные компоненты:
- **Node.js** версии 18.0 или выше
- **npm** (устанавливается вместе с Node.js)
- **Современный веб-браузер** (Chrome, Firefox, Safari, Edge)

### Рекомендуемые характеристики системы:
- **ОЗУ**: минимум 4 ГБ
- **Свободное место на диске**: минимум 1 ГБ
- **Интернет-соединение** для загрузки зависимостей

## Пошаговая инструкция по установке и запуску

### Шаг 1: Установка Node.js

1. Перейдите на официальный сайт: https://nodejs.org/
2. Скачайте LTS версию для вашей операционной системы
3. Запустите установщик и следуйте инструкциям
4. Проверьте установку, выполнив в терминале:
   ```bash
   node --version
   npm --version
   ```

### Шаг 2: Подготовка проекта

1. Откройте терминал/командную строку
2. Перейдите в папку с проектом:
   ```bash
   cd путь/к/папке/проекта
   ```

### Шаг 3: Автоматическая установка (рекомендуется)

Выполните скрипт автоматического развертывания:

```bash
./deploy.sh
```

Этот скрипт автоматически:
- Проверит системные требования
- Установит все необходимые зависимости
- Соберет приложение для продакшена
- Подготовит файлы для развертывания

### Шаг 4: Ручная установка (альтернативный способ)

Если автоматический скрипт не работает, выполните команды вручную:

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Запуск в режиме разработки:**
   ```bash
   npm run dev
   ```

3. **Сборка для продакшена:**
   ```bash
   npm run build
   ```

## Способы запуска приложения

### Режим разработки (для тестирования и отладки)

```bash
npm run dev
```

- Приложение будет доступно по адресу: http://localhost:3000
- Автоматическая перезагрузка при изменении кода
- Подробные сообщения об ошибках

### Продакшен режим (для развертывания)

1. **Сборка приложения:**
   ```bash
   npm run build
   ```

2. **Запуск локального сервера:**
   ```bash
   npx serve -s build -l 3000
   ```

3. **Развертывание на веб-сервере:**
   - Скопируйте содержимое папки `build/` на ваш веб-сервер
   - Настройте веб-сервер для обслуживания статических файлов

## Основные функции приложения

### 1. Экран статистики
- Отображение общего количества контактов
- Круговая диаграмма качества контактных данных
- Детальная разбивка по полям контактов
- Индикаторы заполненности основных полей

### 2. Экран выбора контактов
- Поиск контактов по имени, телефону или организации
- Модальное окно для выбора контакта
- Фильтрация результатов поиска в реальном времени

### 3. Экран досье контакта
- Отображение основной информации о контакте
- Интеграция с публичными профилями (LinkedIn, Facebook, Twitter)
- Возможность связывания профилей с контактом
- Импорт данных из социальных сетей

### 4. Экран категорий
- Организация контактов по категориям (Семья, Друзья, Коллеги и т.д.)
- Подкатегории для более детальной классификации
- Управление неклассифицированными контактами
- Статистика по категориям

## Устранение неполадок

### Проблема: "command not found: node"
**Решение:** Установите Node.js с официального сайта nodejs.org

### Проблема: "npm ERR! peer dep missing"
**Решение:** Удалите папку `node_modules` и файл `package-lock.json`, затем выполните `npm install`

### Проблема: Порт 3000 уже занят
**Решение:** Используйте другой порт: `npm run dev -- --port 3001`

### Проблема: Ошибки при сборке
**Решение:** 
1. Проверьте версию Node.js (должна быть 18+)
2. Очистите кэш: `npm cache clean --force`
3. Переустановите зависимости: `rm -rf node_modules package-lock.json && npm install`
